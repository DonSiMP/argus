# dispair
Stereo odometry using dense **dis**parities generated by **pair**s of cameras. Wraps Hatem Ismail's [bpvo](https://github.com/halismai/bpvo) dense visual odometry library to use argus conventions and displacement prediction.

# Dependencies
* [argus_utils](https://github.com/Humhu/argus_utils): Utilities for all argus packages

# Class Overview
## ImageSync
Wraps exact or approximate image and info synchronization.

# Node Overview
## disparity_vo_node
Estimates camera motion from consecutive grayscale and disparity images.
